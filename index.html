Install in four steps

Takes about a minute once the download finishes.

1

Download the pack

Use the button above. It is one zip with every clothing preview image, the item name data, and the icons.

2

Open your Cherax folder

That is the folder containing Lua and Cache. For most people it is here:

Documents\Cherax
3

Extract the zip into that folder

Right click the zip, choose Extract All, and set the destination to your Cherax folder. If Windows asks about merging folders or replacing files, choose yes to all.

4

Reload LiquidFit

Reload the script in the Lua tab (or restart Cherax). Previews appear instantly for every category.

Ended up with a LiquidFit_PreviewPack folder? Your extractor made a wrapper folder. Open it, then move the inner Cache and Lua folders up into the Cherax folder so they merge with the existing ones.

Quick check: Cherax\Cache\LiquidFit\thumbs should contain thousands of images.

The pack is optional. LiquidFit works without it, items just show as tiles without pictures.
Get the script itself on the Cherax script hub.
